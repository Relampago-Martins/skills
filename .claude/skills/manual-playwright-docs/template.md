## Notas

- Antes de escrever o manual final, monte um roteiro curto e valide-o quando o pedido vier com pouco contexto.
- Prefira um arquivo principal por manual.
- Se o manual passar de ~250-300 linhas, separe em múltiplos arquivos por módulo.
- Use screenshots focadas para tabelas, menus, filtros, cards e formulários.
- Use screenshots amplas apenas para abrir uma seção ou dar contexto.
